## 1. 프로젝트 개요

경기도 교육행정직 공무원의 반복 업무를 자동화·간소화하는 실용 도구 모음 웹사이트.  
강원도 교행노트(gyohaeng.pages.dev)를 참고하여 경기도 버전으로 제작.

### AI 도구 역할 분담
| 도구 | 역할 |
|------|------|
| Claude | 기획·설계·문서화·코드 리뷰·막히는 개념 질문 |
| Cursor | 실제 코드 편집 (메인 IDE) |
| Codex | 반복 코드 초안 생성 |

## 2. 기술 스택 및 환경

### 프론트엔드
- React + Vite
- Tailwind CSS
- React Router (페이지 라우팅)
- recharts (차트)
- xlsx / exceljs (엑셀 파싱 및 생성)

### 배포
- GitHub → Cloudflare Pages 자동 배포
- push하면 자동으로 사이트 업데이트

### 환경 변수
- 로컬: `.env` 파일에 `VITE_NEIS_API_KEY=실제키값`
- 배포: Cloudflare Pages → Settings → Variables and Secrets 등록
- `.env`는 `.gitignore`로 GitHub 업로드 차단

### 주요 설정
- `wrangler.toml` 삭제 (대시보드 환경변수 관리 위해)
- NEIS API: 브라우저에서 직접 호출 (CORS 허용 확인)
- `public/_redirects`: SPA 라우팅용

### 🏫 학교 정보 조회 (`/school-info`)
- **상태:** 사용가능
- NEIS Open API 활용
- 학교 이름 검색 → 목록에서 선택
- 이번달 학사일정 월간 달력 표시
- 날짜 클릭 → 해당 날짜 중식 급식 표시
- 마지막 검색 학교 localStorage 자동 저장

**구현 특이사항:**
- Cloudflare Pages IP가 NEIS 서버에서 차단됨
- 프록시 방식 → 브라우저 직접 호출로 변경
- NEIS API CORS 허용 확인 (`Access-Control-Allow-Origin: *`)

---

### 💳 카드 고지서 매칭 (`/card-match`)
- **상태:** 사용가능
- 에듀파인 원인행위 목록 + 카드사 청구내역 대조
- 파일 업로드 (드래그앤드롭 지원)
- 브라우저에서만 처리 (서버 전송 없음, 개인정보 보호)
- 결과 엑셀 다운로드 (`카드대조_완성판.xlsx`)
- 사용방법 모달 + 이미지 안내 (BC카드, 에듀파인 캡처)

**지원 형식:** CSV, xlsx, xlsm, xls (첫 번째 시트)

**주요 파일:**
- `src/lib/cardMatch/parseWorkbook.js` - CSV·엑셀 파싱
- `src/lib/cardMatch/matchEngine.js` - 대조 알고리즘
- `src/lib/cardMatch/buildExcel.js` - 결과 엑셀 생성
- `src/pages/CardMatch.jsx` - UI

---

### 🔢 경제적 수리한계 계산기 (`/calculator`)
- **상태:** 사용가능
- 취득가격, 내용연수, 사용연수 입력
- 자동 계산 및 결과 표시

**계산 로직:**
```
사용연수 = 1          → 최초연도: 취득가격 × 70%
사용연수 >= 내용연수  → 최종연도: 취득가격 × 20%
그 외               → 중간연도: (취득가격 × 0.7) - (사용연수 × 취득가격) / (내용연수 × 2)
```

**불용처분 안내:**
- 내용연수 초과 → "수리비와 관계없이 즉시 불용처분 가능"
- 수리한계 초과 → "경제적 수리한계 초과로 불용처분 가능"
- 미초과 → "수리비가 N원을 초과하면 불용처분 가능"

---

### 📦 물품대장 (`/inventory`)
- **상태:** 사용가능
- G2B 물품목록 엑셀 업로드 (헤더 자동 탐지)
- 내용연수 상태별 분류 (초과🔴/임박🟡/여유🟢)
- 경제적 수리한계 자동 계산 (물품 클릭 시 모달)
- 검색 (물품명, 설치장소, 운용부서, 물품고유번호)
- 운용부서 필터 체크박스
- 상태 필터 체크박스 (초과/임박/여유)
- 컬럼 헤더 클릭 정렬 (기본: 내용연수 만료일 오름차순)
- 페이지네이션 (100개씩, 페이지 번호 버튼)
- 잔여기간 표시 (예: 2년 5개월 / 8개월 초과)
- 내용연수 만료일 표시 (색상 구분)
- 현재 목록 엑셀 내보내기 (색상 서식 포함)
- 불용처분 대상 목록 다운로드

**엑셀 파일 구조 파악:**
- 1행: 제목, 2~3행: 빈 행, 4행: 헤더, 5행~: 데이터
- 내용연수: `내용연수(변경)` 우선, 없으면 `내용연수(취득)` 적용
- 사용연수: 오늘 날짜 기준 자동 계산 `(오늘 - 취득일) / 365.25`

---

### 🗂 편철 표지 (`/archive`)
- **상태:** 사용가능
- 지출증빙서 측면 표지 제작 및 인쇄
- 엑셀 기초자료 시트처럼 표 형태 입력 (인라인 편집)
- Tab/Enter로 셀 이동
- 1번 행 기관명 입력 → 나머지 행 자동 동기화
- 인쇄: 새 창으로 열어 A4 가로 출력
- localStorage 저장 (새로고침해도 유지)

**초기값 자동 설정:**
- 제목: "학교회계 지출증빙서"
- 회계연도: 오늘 날짜 기준 자동 (1~2월 → 전년도, 3~12월 → 당해연도)
- 연월: 오늘 날짜 기준 자동
- 기간/일련번호: placeholder만 표시
- 너비 기본값: 3.5cm
- 시작 행: 5개

**출력물 구조 (10행):**
1. 회계연도 (라벨)
2. 회계연도 값
3. 연월 (라벨)
4. 연월 값
5. 기간 값
6. 일련번호 (라벨)
7. 일련번호 값
8. 제목 (세로 큰 글씨)
9. 기관명 (라벨)
10. 기관명 값

## 6. 트러블슈팅 기록

### NEIS API 500 오류
- **원인:** Cloudflare Pages 서버 IP를 NEIS가 차단
- **해결:** 프록시 제거, 브라우저에서 직접 호출

### Cloudflare 환경변수 적용 안 됨
- **원인:** `wrangler.toml` 존재 시 대시보드 Plaintext 변수 무시
- **해결:** `wrangler.toml` 삭제

### 물품대장 엑셀 파싱 실패
- **원인:** 헤더가 4행에 있고 1~3행은 제목/빈행
- **해결:** 헤더 자동 탐지 로직 (빈 행 건너뛰기)

### 편철 표지 인쇄 페이지 분리
- **원인:** `window.print()` 시 CSS `@page` 설정 무시
- **해결:** `window.open()`으로 새 창 열어 인쇄 전용 HTML 생성

### 북마클릿 드래그 설치 안 됨
- **원인:** 크롬 보안 정책으로 `javascript:` 링크 실행 차단
- **해결:** 코드 복사 버튼 + 직접 북마크 추가 방법으로 안내 변경
