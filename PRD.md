## 1. 프로젝트 개요

### 1-1. 배경 및 필요성

경기도 내 교육행정직 공무원들은 매년 반복되는 업무(학사일정 확인, 카드 고지서 처리, 각종 행정 업무)를 개인의 역량에 의존해 처리하고 있습니다. 체계적인 디지털 도구 없이 Excel, 수기 확인에 의존하는 경우가 많아 비효율이 발생합니다.

> 💡 **기획 의도**
> 강원도 교육행정직 공무원이 자체 제작한 ['교행노트'](https://gyohaeng.pages.dev/#home) 사이트처럼,
> 경기도 교육행정직 공무원을 위한 실용 도구 모음 사이트를 구축한다.
> 현장 경험을 바탕으로 실제로 쓰이는 도구를 직접 만들어 동료들에게 공유한다.

### 1-2. 목표
- 경기도 교육행정직 공무원 대상 실용 업무 도구 공유 웹사이트 구축
- 반복 업무를 자동화·간소화하는 프로그램 개발 및 배포
- 현장에서 즉시 쓸 수 있는 심플한 UI/UX 제공
- 오픈 후에도 지속적으로 기능 추가 및 개선 및 업데이트 예정

### 1-3. 사용자
- **주 대상**: 경기도 교육청 소속 교육행정직 공무원
- **부 대상**: 전국 교육행정직 공무원 (범용 기능 한정)

## 2. PRD (제품 기획서)

### 2-1. 사이트 개요

| 항목 | 내용 |
|------|------|
| 사이트명 | 경기교행 업무도우미 |
| 도메인 (예정) | gyeonggi-edu-admin.pages.dev (Cloudflare Pages 무료) |
| 디자인 방향 | 노션 스타일 — 흰 배경, 깔끔한 타이포, 심플한 카드 레이아웃 |
| 기술 스택 (사이트) | React + Tailwind CSS → Cloudflare Pages 배포 |
| 기술 스택 (도구) | React + FastAPI + SQLite (기존 업무캘린더 스택 유지) |

### 2-2. 주요 기능 (MVP)

> MVP = 최소 기능 제품. 처음부터 완벽하게 만들지 않고, 핵심 기능만 먼저 공개하는 전략.

#### 🗓 도구 1: 교육행정직 업무 캘린더
- 학사일정 자동 연동 (NEIS Open API)
- 월간/주간 뷰 전환
- 칸반(Kanban) 스타일 업무 관리
- 학교별 개인 설정 저장
- 기존 개발 중인 React + FastAPI + SQLite 앱을 사이트에 소개 및 다운로드 링크 제공

#### 💳 도구 2: 카드 고지서 매칭 프로그램
- 카드 사용 내역 엑셀/CSV 업로드
- 예산 항목과 자동 매칭
- 불일치 항목 하이라이트
- 결과 엑셀 다운로드
- 웹 기반 (브라우저에서 바로 실행, 서버 불필요 — 개인정보 보호)

### 2-3. 사이트 구조 (페이지 설계)

| 페이지 | 내용 |
|--------|------|
| 홈 (Home) | 사이트 소개, 도구 카드 목록, 최근 업데이트 |
| 업무 캘린더 | 소개 + 다운로드 + 사용법 |
| 카드 고지서 매칭 | 웹 앱 직접 실행 |
| 공지사항 / 업데이트 | 버전 히스토리, 공지 |
| 정보 / 문의 | 제작자 소개, 연락처 |

### 2-4. 비기능 요구사항
- 무료 배포 (Cloudflare Pages — 무료 플랜으로 충분)
- 모바일 반응형 지원 (스마트폰에서도 깨지지 않는 레이아웃)
- 개인정보 수집 없음 (회원가입 불필요)
- 한글 폰트 최적화 (Pretendard 또는 Noto Sans KR)

## 4. 도구별 개발 상세

### 4-1. 교육행정직 업무 캘린더

> **현황**
> - 기술 스택: React + FastAPI + SQLite, NEIS Open API 연동 완료
> - 개발 상태: 핵심 기능 개발 중 (월간/주간 뷰, 칸반 보드)
> - 배포 형태: 실행 파일(.exe) 또는 웹 접속

**남은 개발 항목**
- 필요한 기능 지속적인 개선 및 추가
- 실행 파일 패키징 (PyInstaller)
- 사이트에 소개 페이지 + 다운로드 링크 연결

### 4-2. 카드 고지서 매칭 프로그램

> **개발 방향**
> - 서버 없이 브라우저에서만 동작 (개인정보 외부 유출 없음)
> - 엑셀/CSV 업로드 → 자동 매칭 → 결과 다운로드
> - React 기반 단일 페이지 앱으로 구현

**주요 기능 설계**
- 이미 파일 만들어놨음.

## 5. 배포 전략

### 5-1. 사이트 배포
- 플랫폼: Cloudflare Pages (무료, 빠름, HTTPS 자동)
- 도메인: 기본 제공 도메인 사용 → 추후 커스텀 도메인 연결 가능
- GitHub 연동: push하면 자동 배포

> **배포 흐름**
> ① Cursor에서 코드 작성
> ② GitHub에 push
> ③ Cloudflare Pages가 자동으로 사이트 업데이트
> → 별도 서버 관리 불필요, 비용 0원

### 5-2. 홍보 전략
- 교육행정직 커뮤니티 (에듀넷, 직장인 카페) 공유
- 경기도교육청 내부 메신저/메일로 동료 공유
- 사용 후기 수집 → 기능 개선 사이클 운영
